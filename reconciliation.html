The ETL system must be able to recognize that the ﬁlehas appeared in the ﬁle system and execute automatically. This process is called token aware. Tokens are ﬁles created in the ﬁle system to trigger an ETL event. Applications that are token aware can poll a directory (or database table) for the arrival of a token ﬁle (or a row).
